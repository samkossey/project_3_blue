Command-Line Minesweeper

Users can play minesweeper in their command-line by entering desired board dimensions and a number of bombs. The program creates a board that fits the user's specifications. Continued game play involved deciding to uncover or flag different tiles until either a bomb is chosen or the entire board has been uncovered aside from the bombs.

How it Works

At its core, this program relies on a 2d array to represent the game board. Important features are the randomization of the bombs, the calculations for the number of adjacent bombs given a tile, and the termination conditions of the game. Another important functionality comes when a user uncovers a tile. The board should also uncover adjacent blank tiles and a border of numbered tiles after this action, as per the rules of minesweeper.

Security Considerations

Evaluation of code security was performed based on the Arrays (ARR) and Expressions (EXP) rules in the SEI CERT C/C++ Coding Standard. These rules were considered throughout the development stage as well to gain a better understanding of how to use them in practice.
